# FAQ

## Is Docs Hub based on Material?

No. Docs Hub is a custom MkDocs theme.

## Does it support search?

Yes. It uses the MkDocs search plugin and a custom search UI.

## Does it support tabs and callouts?

Yes. Docs Hub supports:

- admonitions
- expandable details
- tabbed content

## Can I customize the colors?

Yes. The theme uses CSS variables, so colors and spacing can be adjusted easily.

## Which page should I use to test changes?

Use a dedicated `components.md` page so you can verify all UI patterns in one place.

## What should I check before publishing?

Before a release, verify:

- navigation
- search
- typography
- code blocks
- admonitions
- tabs
- details
- mobile layout
